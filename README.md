# utils-container
An ubuntu based image (23.04) used for troubeshooting tasks.
Includes: 
- bash
- bridge-utils
- cifs-utils
- crictl (from https://github.com/kubernetes-sigs/cri-tools)
- curl
- dnsutils
- grpcurl (from https://github.com/fullstorydev/grpcurl)
- iotop
- iperf
- iptables
- iputils-ping
- jq
- net-tools
- netcat-traditional
- nmap
- procinfo
- procps
- smbclient
- socat
- stress
- stress-ng
- sysbench
- systemd
- tcpdump
- vim
- wget
